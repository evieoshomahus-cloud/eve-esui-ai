## Colour Direction

The current ESUI website uses a strong blue overlay in its hero section: `#203B94`. The Eve interface uses this as the primary institutional colour. Supporting colours were chosen from the website and logo direction:

- Primary institutional blue: `#203B94`
- Supporting blue: `#2C5AA0`
- Deep blue: `#1E3D6F`
- Logo green accent: `#79B900`
- Operational gold/status accent: `#E2A03F`
- Interface background: `#F7F9FD`

## UI Direction

The redesigned interface starts with a lightweight session entry screen, then moves into a personalized operational home screen. This makes Eve feel like an official campus companion rather than a developer dashboard.

The interface is responsive:

- Desktop: navigation rail, personalized home, focused chat workspace, and role-specific tools.
- Mobile/tablet: entry screen, bottom navigation, full chat, and touch-friendly action tiles.
- Defense mode: the web version can be projected from a browser while the same Flutter codebase remains suitable for Android deployment.
